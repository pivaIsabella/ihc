Sobre Kotlin

Kotlin é uma linguagem de programação estática, multiplataforma e de tipagem estática, desenvolvida pela JetBrains em 2011. Ela é compatível com a Java Virtual Machine (JVM) e pode ser usada para desenvolvimento de aplicativos Android, aplicativos desktop e servidores.

Características de Kotlin

  • Interoperabilidade com Java
  • Simplicidade e concisão
  • Nulidade segura
  • Funções de extensão
  • Lambdas
  • Programação assíncrona com coroutines
